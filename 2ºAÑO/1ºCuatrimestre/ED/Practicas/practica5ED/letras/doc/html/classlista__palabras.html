Descripción detallada

T.D.A. lista_palabras.

Una instancia lista del tipo de datos abstracto lista_palabras es un objeto del tipo set compuesto por varias palabras sobre el tipo string Lo representamos como;

lista_palabras = set<string> palabras siendo cada elemento(palabras.elemento) una palabra de la lista de palabras del set
