Descripción detallada

T.D.A. conjunto_letras.

Una instancia conjunto del tipo de datos abstracto conjunto_letras es un objeto del tipo set compuesto por varias letras sobre el tipo letra Lo representamos como;

conjunto_letras = set<letra> conjunto siendo cada elemento(conjunto.elemento) una letra del set compuesto por letras
