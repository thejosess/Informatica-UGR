T.D.A. letra.

Una instancia c1 del tipo de datos abstracto letra es un objeto del tipo set compuesto por varias letras sobre el tipo letra Lo representamos como;

letra = char, int, puntuacion siendo el char el valor de la letra, el int el numero de veces que aparece y otro int con la puntuacion
