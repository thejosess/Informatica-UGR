Descripción detallada

template<typename T>
class Cola_max< T >

T.D.A. Cola_max.

Se trata de una cola con dos implementaciones, una realizada a partir de un vector y otra a partir de dos pilas, pero ambas implementaciones funcionan de igual manera. Esta cola guarda un pair que contiene el elemento que corresponde con esa posición de la cola y el elemento máximo que hay actualmente en la cola.
